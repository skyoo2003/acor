# Support

## Getting Help

If you need help with ACOR, here are the available channels:

### Documentation

- **User Guide**: [GitHub Pages](https://skyoo2003.github.io/acor/)
- **API Reference**: [pkg.go.dev](https://pkg.go.dev/github.com/skyoo2003/acor)

### Community

- **Questions**: [GitHub Discussions](https://github.com/skyoo2003/acor/discussions) — Ask questions, share ideas, and connect with other users
- **Bug Reports**: [GitHub Issues](https://github.com/skyoo2003/acor/issues/new/choose) — Report bugs using the bug report template
- **Feature Requests**: [GitHub Issues](https://github.com/skyoo2003/acor/issues/new/choose) — Suggest new features using the feature request template

### Response Times

| Channel | Expected Response |
| --- | --- |
| Bug reports | 48 hours |
| Feature requests | 7 days |
| GitHub Discussions | Best effort |

## Reporting Security Issues

**Do not report security vulnerabilities through public issues.**

Please report them privately through [GitHub Security Advisories](https://github.com/skyoo2003/acor/security/advisories/new). See [SECURITY.md](SECURITY.md) for details.

## Professional Support

ACOR is a community-driven open-source project. There is no commercial support available at this time. If your organization requires professional support, consider sponsoring the project through [GitHub Sponsors](https://github.com/sponsors/skyoo2003).
